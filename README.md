# __Примеры использования основных функций__

__1. Начало работы__

__- Откройте приложение двойным кликом мыши__

__- Выберите вкладку "Настройки" и настройте основные параметры (громкость, звук)__

__2. Установка будильника__

__- Нажмите на кнопку "+ Добавить сигнал"__

__- Укажите точное время срабатывания и выберите тип уведомления ("мягкий звонок", "мелодия", "звук тревоги")__

__3. Повтор сигнала__

__- Установите периодичность повторения ("ежедневно", "каждый рабочий день", "разово")__

*Можно задать количество повтора сигнала перед отключением*

__4. Работа с группами сигналов__

__- Создайте новую группу сигналов (например, "Работа", "Отдых")__

__- Переместите сигналы в нужную группу для удобства сортировки__

__5. Экспорт/импорт настроек__

__- Экспортируйте настройки своего будильника для переноса на другое устройство__
__- Импортируйте ранее сохраненные настройки другим пользователям__
